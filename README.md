# GIS Project

## Installation

```bash
  git clone https://github.com/alpeer/gis-project.git
  cd gis-project/frontend
  npm install
```

## Commands

### Start dev server

```bash
  npm start
```

### Production Build

```bash
  npm run build
```

### Analyze

Builds and generates source-map, launches analyzer

```bash
  npm run analyze
```

### Serve

```
  npm run build
  npm run serve
```

#### Test

```bash
  npm test
```
